-->>> LangChain is a framework that helps you turn an LLM into a complete AI application.
An LLM alone can only answer based on what it already knows.

LangChain helps the LLM:

Read PDFs and documents
Search a database
Use APIs and tools
Remember previous conversations
Perform tasks step by step
Build RAG applications and AI agents


Easy example

Suppose you are building a hospital chatbot.

The user asks:

“What was my last prescription?”

A normal LLM does not know the patient’s personal prescription.

With LangChain, the process can be:

User asks a question
↓
LangChain checks the patient database
↓
It finds the relevant prescription
↓
It sends that information to the LLM
↓
The LLM gives a clear answer


---------->>>>>> MIDDLEWARE


what is middleware in LangChain?

Middleware is extra logic that runs before, after, or around an AI agent’s actions.

--->> Built-in middleware

Built-in middleware is already created by LangChain.

Common built-in middleware includes:

SummarizationMiddleware — summarizes long conversations
HumanInTheLoopMiddleware — asks a human before sensitive actions


---->>> Custom middleware

Custom middleware is middleware that you create yourself for your project’s specific requirements.

You can create custom middleware for:

Custom logging
Checking user permissions
Blocking specific words
Adding user information to the prompt
Tracking token usage
Choosing a different model based on the request
Limiting conversations
Custom error handling
